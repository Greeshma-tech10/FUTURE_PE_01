# NeighbourHUB — AI Website Copy Generator for Apartment Communities

## About the Project

NeighbourHUB is a smart apartment community platform designed to simplify apartment living through visitor management, resident engagement, maintenance operations, home services, and local marketplace features.

The platform combines security, communication, and local services into one connected ecosystem for apartment societies and gated communities.

---

## Objective

The objective of this project is to design a reusable AI prompt engineering system capable of generating professional SaaS website copy for apartment community platforms.

This project focuses on:
- AI-generated website copy
- Prompt engineering
- SaaS landing page content
- CTA optimization
- Modular content generation
